## 新手入门建议

1. **打好编程基础**：至少熟练掌握一门语言，推荐 Python（写脚本方便）和 C（理解底层）。
2. **学习 Linux**：绝大部分安全工具和服务器环境都在 Linux 上运行，熟悉命令行是基本功。
3. **理解网络协议**：TCP/IP、HTTP、DNS 等，这是 Web 安全和流量分析的基础。
4. **选择一个方向深入**：不要贪多，先集中精力在一个方向，比如二进制或 Web。
5. **参加 CTF 比赛**：实践出真知，从简单的签到题开始，积累经验。

## 推荐资源

- **在线平台**：CTFhub、BUUCTF、攻防世界
- **书籍**：《深入理解计算机系统》（CSAPP）、《0day安全：软件漏洞分析技术》
- **工具**：GDB + pwntools（二进制）、Burp Suite（Web）、Ghidra（逆向）
